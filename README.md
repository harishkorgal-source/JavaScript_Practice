# JavaScript Practice

A collection of JavaScript exercises covering fundamental concepts, organized by topic.

## Topics Covered

| # | Topic | Description |
|---|-------|-------------|
| 01 | Basic | Basic JavaScript syntax and setup |
| 02 | Variable | Variable declarations (`var`), redeclaration, and function basics |
| 03 | Operator | Ternary operator with conditional expressions |
| 04 | If-Else | Conditional statements — even/odd checker, grade calculator |
| 05 | Arrays | Array fundamentals and manipulation |

## Playwright

Browser automation and end-to-end testing with [Playwright](https://playwright.dev/).

| Test File | Description |
|-----------|-------------|
| example.spec.ts | Verifies Playwright docs site title and navigation |
| login.spec.ts | Login flow automation for Amazon |

### Running Playwright Tests

```bash
cd Playwright
npx playwright test
```

## How to Run

```bash
node <filename>.js
```

Example:

```bash
node 03_Operator/Ternary.js
```

## Prerequisites

- [Node.js](https://nodejs.org/) installed on your machine
- [Playwright](https://playwright.dev/) for browser automation tests
