# JavaScript Practice

A collection of JavaScript exercises covering fundamental concepts, organized by topic.

## Topics Covered

| # | Topic | Description |
|---|-------|-------------|
| 01 | Basic | Basic JavaScript syntax and setup |
| 02 | Variable | Variable declarations (`var`), redeclaration, and function basics |
| 03 | Operator | Ternary operator with conditional expressions |
| 04 | If-Else | Conditional statements — even/odd checker, grade calculator |

## How to Run

```bash
node <filename>.js
```

Example:

```bash
node 03_Operator/Ternary.js
```

## Prerequisites

- [Node.js](https://nodejs.org/) installed on your machine
